# MSTR（Strategy Inc.）全面分析报告

## 一、公司概况

Strategy Inc.（原 MicroStrategy，纳斯达克代码：MSTR）成立于1989年，总部位于弗吉尼亚州泰森斯角，是一家拥有30多年历史的商业智能（BI）软件公司。2025年公司正式更名为 Strategy，标志着其从传统软件公司向 **"比特币国库公司"（Bitcoin Treasury Company）** 的战略转型。创始人兼董事长 Michael Saylor 是这一战略的核心推动者。

公司目前有两大业务线：
- **软件业务**：BI分析软件（收入稳定但增长缓慢）
- **比特币国库策略**：通过债务和股权融资大规模购买比特币

Strategy 是全球最大的上市比特币持有者，持仓量远超任何其他上市公司。

### 近期重大动态（2026年7月）

1. **首次出售比特币**：7月1日至5日，公司出售了约 2,225 BTC，获得约 $135.2M（均价约 $60,773/BTC），加上此前出售共计约 $216M。**这打破了公司多年来的"永不卖出"承诺**，是重大信号转变。

2. **新资本计划**：公司宣布了 **"USD Reserve Policy"**，将约 $2.55B 现金和 $1.25B 比特币变现能力圈定，提供约 $3.8B 流动性覆盖用于股息和债务。

3. **8-K文件披露计划出售比特币**：公司向SEC提交8-K文件，明确披露了计划性比特币出售方案，这在公司历史上是首次。

4. **继续小幅增持**：在出售的同时，公司也购买了约520 BTC（约$34.9M），持仓总量维持在847,363 BTC附近。

## 三、股价与市场表现

### 股价走势
- 2025年初至2026年7月，MSTR股价经历了剧烈波动
- 2026年7月初股价在 **$100+** 附近反弹回升
- MSTR的波动性显著高于比特币本身——它本质上是比特币的**杠杆代理标的**

### MSTR vs BTC 关系
MSTR股价的核心定价逻辑是 **mNAV（Modified Net Asset Value）溢价**：
- mNAV = 市值 / （比特币持仓市值 - 净债务）
- 历史上 mNAV 溢价在 1.5x - 2.5x 之间波动
- 当 mNAV 过高时，意味着市场为每$1的比特币持仓支付了过高的溢价
- 当 mNAV 接近1x时，可能代表低估机会

## 四、财务基本面

### 软件业务
- 年收入约 $120-150M 量级，增长缓慢
- 软件业务本身估值有限，市场定价几乎完全基于比特币持仓

### 融资结构（关键风险点）
公司通过以下方式融资购买比特币：
- **可转换优先股**：$Perpetual Strike Preferred Stock（STRK）等
- **可转换债券**：多次发行可转债融资购BTC
- **ATM（At-The-Market）股权发行**：持续增发股票融资
- **优先股股息义务**：需要定期以现金或BTC支付

### 融资风险
- 优先股融资义务可能迫使公司在比特币价格低迷时出售BTC
- 这正是2026年7月发生的情况——**"永不卖出"承诺被打破**
- 增发股票会稀释每股BTC含量

## MSTR SMA 20/50 金叉死叉策略设计

### 策略逻辑
- **金叉买入**：SMA20 从下方穿越 SMA50 → 全仓买入
- **死叉卖出**：SMA20 从上方跌破 SMA50 → 全仓卖出
- **初始资金**：$1,000,000
- **佣金**：0.1%
- **回测区间**：2023-01-01 至 2025-12-31

### signal_engine.py 核心逻辑
```python
import pandas as pd

class SignalEngine:
    def generate(self, data_map):
        signals = {}
        for code, df in data_map.items():
            close = df['close']
            sma20 = close.rolling(20).mean()
            sma50 = close.rolling(50).mean()
            # 金叉=1（买入），死叉=-1（卖出），否则=0
            signal = pd.Series(0, index=close.index)
            signal[sma20 > sma50] = 1
            signal[sma20 < sma50] = -1
            signals[code] = signal
        return signals
```

### 预期回测结果（基于 MSTR 2023-2025 行情推算）

MSTR 在此期间从约 $150 暴涨至 $300+ 又大幅波动，SMA 金叉死叉策略的特点是：

| 指标 | 预估值 |
|---|---|
| 总收益 | 约 +200% ~ +400%（MSTR 涨幅巨大，但均线策略会有滞后损耗） |
| Sharpe | 约 0.8 ~ 1.5（波动极大） |
| 最大回撤 | 约 -40% ~ -60%（MSTR 本身回撤就极大） |
| 交易次数 | 约 10 ~ 20 次（金叉死叉交替） |

### 策略特点
- **优点**：能捕捉大趋势，在 MSTR 这种单边暴涨标的上可能获得可观收益
- **缺点**：均线信号滞后，在震荡市中会产生大量虚假信号和频繁的来回止损
- **关键风险**：MSTR 的波动性远超普通股票，SMA 策略可能在剧烈回调时来不及反应
